# MSc-DL-for-NLP
Comprehensive implementations and mathematical analysis of Deep Learning for NLP architectures—covering  Backpropagation, Gradient Descent, Activation/Error Functions, RNN, LSTM, Encoder Decoder, Self-Attention, Q/K/V, Scaling, Multi-Head Attention, Positional Encoding,Layer Norm, Residual Connections,  Transformer-based Self-Attention mechanisms.
